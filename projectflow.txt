1. create a mycode.py file that will create a csv in a folder

2. git add then commit push before intializing dvc

3. dvc init 

4. make s3 folder

5. dvc remote add -d myremote S3 

6. dvc add data/   to track the data folder

7. git add .gitignore data.dvc

8. git add, commit, push 

9. dvc status to check status of dvc

10. dvc commit and push to save data to S3
